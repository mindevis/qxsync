# QXSync

QX / QM Minecraft mod for **NeoForge**. Built and released from this repository.

**Release:** push to `main` with [Conventional Commits](https://www.conventionalcommits.org/) — [semantic-release](https://github.com/semantic-release/semantic-release) publishes JARs to GitHub Releases.

Related: [qm-project](https://github.com/mindevis/qm-project) monorepo (launcher & stack).

## Releases

Pushes to `main` with a `fix:` or `feat:` commit trigger [semantic-release](https://github.com/semantic-release/semantic-release); JARs are attached to GitHub Releases.
